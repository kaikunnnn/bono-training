# 記事詳細ページ - Heading Section デザイン仕様書

## 📋 概要

記事詳細ページの最上部に配置される、記事のタイトル、説明、クエスト情報、アクション（完了・お気に入り・シェア・次へ）を表示するセクションです。

---

## 🎯 構造概要

```
heading-section（幅720px）
├── quest_info（クエスト情報）
├── title（タイトル）
├── description_text（説明文）
└── actionarea（アクションボタン領域）
    ├── left（左側ボタングループ）
    │   ├── check_button（完了ボタン）
    │   ├── favorite_button（お気に入りボタン）
    │   └── share_button（シェアボタン）
    └── button_next_content（次コンテンツボタン）
```

---

## 📐 全体レイアウト

| プロパティ           | 値                          |
| -------------------- | --------------------------- |
| **コンポーネント幅** | 720px（固定）               |
| **レイアウトモード** | Column（縦並び）            |
| **ギャップ**         | 12px                        |

## 🔤 Section 1: クエスト情報（quest_info）

### レイアウト情報

| プロパティ           | 値                     |
| -------------------- | ---------------------- |
| **レイアウトモード** | Row（横並び）          |
| **ギャップ**         | 4px                    |
| **サイズ**           | Hug（自動）            |
| **配置**             | heading-section の上部 |

### 構成要素

#### 1-1. クエスト番号テキスト

- **要素**: `numberofquest`
- **テキスト**: `クエスト2`（サンプル）
- **フォント**: Inter
- **フォントサイズ**: 12px
- **フォントウェイト**: 400（Regular）
- **行高**: 24px（2em）
- **文字間**: -2.604%
- **色**: #747474（グレー）
- **配置**: LEFT / TOP

#### 1-2. スラッシュ区切り文字

- **要素**: `/`
- **テキスト**: `/`
- **フォント**: Inter
- **フォントサイズ**: 12px
- **フォントウェイト**: 400
- **行高**: 24px（2em）
- **文字間**: -2.604%
- **色**: #C0C0C0（ライトグレー）

#### 1-3. ステップ番号テキスト

- **要素**: `numberofcontentinthequest`
- **テキスト**: `ステップ5`（サンプル）
- **フォント**: Inter
- **フォントサイズ**: 12px
- **フォントウェイト**: 400
- **行高**: 24px（2em）
- **文字間**: -2.604%
- **色**: #747474（グレー）

### データ例

```json
{
  "questNumber": 2,
  "stepNumber": 5
}
```

## 🎭 ボタンスタイル詳細

### Button/Secondary（完了ボタン）

- **背景**: #F3F5F5（ライトグレー）
- **テキスト色**: #34373D（濃グレー、bold）
- **状態**: デフォルト（未完了）
- **ホバー状態**: 背景色を濃くするか、透明度 50%→75%に変更を推奨

### Button/Tertiary（お気に入り・シェア）

- **背景**: 透明またはなし（枠線のみ推奨）
- **テキスト色**: #6A7282（ミディアムグレー）
- **状態**: デフォルト
- **ホバー状態**: テキスト色を濃くするか、背景に薄いハイライト追加を推奨

---

## 📱 レスポンシブ対応

### デスクトップ（720px+）

- 幅: 720px（固定）
- アクションエリアは space-between で両端配置
- すべてのボタンが 1 行で表示

### タブレット・モバイル対応推奨

- コンテナ幅を視点幅に合わせ調整（max-width: 720px）
- 左ボタングループが改行する場合は wrap 設定
- 「次の動画」ボタンは 1 行目に配置、またはスタックして表示

---

## 🔄 インタラクション設定

### 完了ボタン

- **クリック**: 完了状態をトグル
- **状態遷移**: completed: false → true
- **UI 更新**: ボタンの見た目を「完了済み」に変更（チェックマーク強調など）

### お気に入りボタン

- **クリック**: お気に入り状態をトグル
- **状態遷移**: favorited: false → true
- **UI 更新**: 星アイコンを塗りつぶし状態に変更

### シェアボタン

- **クリック**: シェアダイアログまたはメニューを表示

### 次の動画ボタン

- **クリック**: 次のコンテンツページへナビゲート（contentId 参照）
