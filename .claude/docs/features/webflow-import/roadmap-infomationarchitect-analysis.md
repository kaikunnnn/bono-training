## 2. コンテンツ詳細 (Content Details)

### 2.1 ヒーローセクション

| 項目 | 内容 |
|------|------|
| **コース名** | 情報設計入門コース |
| **完了目安** | 2ヶ月 |
| **説明文** | 使いやすいUIをつくるための基本を習得するコースです。顧客、目的に即したUIを検討→実現する、デザインの組み立て方を身につけます。 |
| **料金** | ￥5,800~/月 |

### 2.2 コース概要

**コンセプト**: 「目的」と「顧客」を中心軸としたUI設計

**学習成果**: 顧客にとって操作しやすいUIアイデアを作る考え方を身につけます

## 9. 再現用データ構造

```typescript
interface RoadmapCourse {
  id: string;
  title: string;                    // "情報設計入門コース"
  estimatedDuration: string;        // "2ヶ月"
  description: string;
  price: string;                    // "￥5,800~/月"
  sections: Section[];
  relatedCourses: RelatedCourse[];
  completionMessage: string;
}

interface Section {
  number: number;                   // 1, 2, 3
  title: string;                    // "つかいやすUIの習得"
  goal: string;                     // "デザインの流れをトレース"
  duration: string;                 // "1-2週間"
  series: Series[];
  options?: ChallengeOption[];      // セクション3のみ
}

interface Series {
  title: string;                    // "つかいやすいUIの秘密"
  tag: string;                      // "#実践して理解"
  description?: string;
  topics?: string[];                // ["モード", "アクション", ...]
  link: string;
}

interface ChallengeOption {
  title: string;
  description: string;
  tag?: string;
  link: string;
}

interface RelatedCourse {
  title: string;
  link: string;
  topics: string[];
}
```

## 10. 実装メモ

### 10.1 必要なコンポーネント

1. **RoadmapHero** - コースタイトル、説明、料金、CTAボタン
2. **RoadmapSection** - アコーディオン展開式セクション
3. **SeriesCard** - シリーズコンテンツカード
4. **ChallengeOption** - 実践お題オプションカード
5. **CourseCompletion** - CLEAR!バッジと完了メッセージ
6. **RelatedCourseCard** - 関連コースカード

### 10.2 インタラクション

- アコーディオン展開/折りたたみ
- ホバーエフェクト（リンク、カード）
