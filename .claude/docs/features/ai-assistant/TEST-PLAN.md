### 🟦 T-05: ロードマップ推薦（核心テスト）

**操作**: 以下をそのままコピペして送信

```
Figmaの操作を一通り勉強し終えて、ポートフォリオに載せる作品を制作中です。
・ポートフォリオに載せる内容は、今作成しているものをブラッシュアップしたものでよいですか。
・サービス全体の全ての画面を作る必要はありますか。
・プロトタイプとして動きを付けたほうが良いですか？
```

**期待される結果**:
- [ ] 「Figmaを学んだ直後にポートフォリオは早い」というような文脈整理がある
- [ ] BONOのロードマップやレッスンへのリンクが含まれる（例: `/roadmaps/...` や `/lessons/...`）
- [ ] リンクがMarkdown形式でクリッカブルになっている

## 問題が起きた場合の確認先

| 症状 | 確認場所 |
|---|---|
| AI応答が返ってこない | Vercel Functions ログ → `api/ai-chat` |
| 「APIエラー」が表示される | `ANTHROPIC_API_KEY` が正しく設定されているか |
| コンテンツが提案されない | `VITE_SANITY_PROJECT_ID` がVercelに設定されているか |
| ページが真っ白 | Vercel Deploymentのビルドログ |
| リンクが404になる | リンク先のslugが実際に存在するか |
