## 変更点まとめ

### 🔴 大きな変更

| 項目 | 現在 | 新デザイン |
|------|------|-----------|
| **レイアウト** | 1カラム（中央寄せ） | 2カラム（サイド+メイン）※デスクトップのみ |
| **ヒーロー** | グラデ背景 + 楕円弧 + 中央配置アイコン | シンプル化（アイコンはサイドへ移動） |
| **サイドバー** | なし | **新規追加**（レッスンカード + ロードマップ）※デスクトップのみ |
| **進捗表示** | QuestCard内のみ | タイトル**下**にプログレスバー + QuestCard内も**併存** |

### 📦 保存コンポーネント

| コンポーネント | パス | 用途 |
|---------------|------|------|
| EllipseArc | `src/components/ui/decorative/EllipseArc.tsx` | 楕円弧装飾（再利用可能） |

※楕円弧はヒーローから削除するが、他の場所で使えるようコンポーネント化して保存

### 🟡 中程度の変更

| 項目 | 現在 | 新デザイン |
|------|------|-----------|
| **戻るボタン** | ヒーロー内（左上） | 独立ヘッダー（左上） |
| **シェアボタン** | なし | ヘッダー右上に追加 |
| **カテゴリ表示** | InfoBlock（ヒーロー内） | タグ形式（タイトル上） |
| **概要リンク** | タブ切り替え | 「概要・目的ですべてみる」リンク追加 |
| **クエストカード矢印** | 個別記事に矢印 | カードヘッダーに矢印（展開示唆） |

### 🟢 小さな変更 / 維持

| 項目 | 状態 |
|------|------|
| タブ構成 | 維持（コンテンツ / 概要・目的） |
| クエスト番号表示 | 維持（クエスト 01, 02...） |
| 縦点線 | 維持 |
| 記事アイテム構成 | ほぼ維持（番号+サムネ+タイトル+時間） |
