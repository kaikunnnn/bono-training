# 画面仕様ドキュメント

進捗機能の画面ごとの仕様を確認するためのファイル群です。

## ファイル一覧

| ファイル | 内容 | ステータス |
|---------|------|-----------|
| [mypage-progress-section.md](./mypage-progress-section.md) | マイページの進行中セクション | 確認待ち |
| [progress-page.md](./progress-page.md) | 進行中画面（一覧） | 確認待ち |
| [article-complete-button.md](./article-complete-button.md) | 記事完了ボタン・トースト | 確認待ち |

## 使い方

1. Figmaや画面イメージのURLを各ファイルに貼り付ける
2. 仕様と照らし合わせて確認事項を記入
3. 確認が取れたら「✅ 確認済み」に更新
