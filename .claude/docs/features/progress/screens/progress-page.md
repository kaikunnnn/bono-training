## 仕様確認

### 画面構成

**進行中セクション**
- [ ] `status != 'completed'` のレッスンを `updated_at` 降順で表示
- [ ] レッスンオブジェクト（進捗率 + 次の記事）を表示
- [ ] 最大表示件数: 制限なし？ or 上限あり？

**一覧セクション**
- [ ] `status = 'completed'` のレッスンを表示
- [ ] 並び順: 100%完了が先、その後 `updated_at` 降順
- [ ] 進捗率のみ表示（「次の記事」は不要）
- [ ] 最大表示件数: 50件程度を想定

### レッスンオブジェクト（進行中セクション）

- [ ] レッスンアイコン
- [ ] レッスンタイトル
- [ ] 進捗バー + 進捗率
- [ ] 次の記事ブロック（"次へ 👉️" + 記事タイトル）

### レッスンオブジェクト（一覧セクション）

- [ ] レッスンアイコン
- [ ] レッスンタイトル
- [ ] 進捗バー + 進捗率
- [ ] 「次の記事」は**不要**

### 100%完了レッスンの操作

- [ ] 「レッスンを完了する」ボタン表示
- [ ] クリックで `status = 'completed'` に更新
- [ ] 進行中から一覧へ移動
