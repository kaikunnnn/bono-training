## 仕様確認

### 完了ボタン

**表示条件**
- [ ] レッスンに属する記事のみ表示
- [ ] レッスン外の記事には**非表示**

**状態**
- [ ] 未完了: ボタン活性
- [ ] 完了済み: チェック状態 or 非活性

### 記事完了時のトースト

- [ ] メッセージ: 「ナイスデザイン 🕺 １歩進んだぜ」
- [ ] 表示時間: ○秒
- [ ] 位置: 画面下部？

### クエスト完了時のトースト

- [ ] メッセージ: 「ナイスすぎぃ！パワぁー！💪」
- [ ] 花火アニメーション
- [ ] 表示時間: ○秒

### レッスン100%完了時のモーダル

- [ ] タイトル: 「おめでとう！新しいスキルを習得しました！」
- [ ] メッセージ: 「レッスンが完了しました。その情熱が旅を前に進めてたんだ。今日は美味しいものでも食べよう」
- [ ] 閉じるボタン or 自動で閉じる
