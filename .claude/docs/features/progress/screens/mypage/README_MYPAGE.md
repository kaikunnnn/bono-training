## 📊 マイページの構成

### ヘッダーエリア
```
マイページ                [アカウント情報] [プロフィール]
─────────────────────────
[すべて] [進捗] [お気に入り] [閲覧履歴]
```

### メインコンテンツ（3つのセクション）

1. **進行中セクション**
   - 見出し「進行中」+ 「すべてみる」
   - サブ見出し「レッスン」
   - ProgressLessonカード × 2（2列並び）

2. **お気に入りセクション**
   - 見出し「お気に入り」+ 「すべてみる」
   - FavoriteArticleCardカード × 4（縦並び）

3. **閲覧履歴セクション**
   - 見出し「閲覧履歴」+ 「すべてみる」
   - ArticleCardカード × 2（縦並び）

## 🎬 タブナビゲーションの動作

### タブの種類

| タブ | 表示内容 |
|------|---------|
| **すべて** | 全セクション表示（進行中 + お気に入り + 閲覧履歴） |
| **進捗** | 進行中セクションのみ表示 |
| **お気に入り** | お気に入りセクションのみ表示 |
| **閲覧履歴** | 閲覧履歴セクションのみ表示 |

### 「すべてみる」リンクの動作

各セクションの「すべてみる」をクリックすると、対応するタブに切り替わります。

例：
- 進行中の「すべてみる」 → 「進捗」タブに切り替え
- お気に入りの「すべてみる」 → 「お気に入り」タブに切り替え

## 📐 重要な数値（クイックリファレンス）

### ページ全体
- 最大幅: `1200px`
- 外側パディング: `40px`（デスクトップ）、`20px`（スマホ）
- コンテンツパディング: `32px`（デスクトップ）、`20px`（スマホ）

### タブナビゲーション
- タブ間ギャップ: `32px`（デスクトップ）、`16px`（スマホ）
- タブフォントサイズ: `16px`
- アクティブ下線: `2px solid #1A1A1A`

### セクション
- セクション間マージン: `56px`
- 見出しフォントサイズ: `20px`
- 見出し下マージン: `20px`
- サブ見出し下マージン: `16px`

### カード配置
- 進行中グリッド: `2列`（デスクトップ）、`1列`（スマホ）
- カード横ギャップ: `24px`
- カード縦ギャップ: `16px`

## 📱 レスポンシブ対応の要点

### デスクトップ（769px以上）
- 進行中セクション: **2列グリッド**
- パディング: 大きめ（40px, 32px）

### スマホ（768px以下）
- 進行中セクション: **1列**に変更
- パディング: 小さめ（20px）
- タブ: 横スクロール可能に

## 💻 実装のポイント

### 1. タブの状態管理

```tsx
const [activeTab, setActiveTab] = useState<'all' | 'progress' | 'favorite' | 'history'>('all');
```

### 2. 条件付きレンダリング

```tsx
{(activeTab === 'all' || activeTab === 'progress') && (
  <ProgressSection />
)}
```

### 3. 2列グリッドの実装

```tsx
<div style={{
  display: 'grid',
  gridTemplateColumns: 'repeat(2, 1fr)',
  gap: '24px',
}}>
  {/* ProgressLessonカード */}
</div>
```

### 4. レスポンシブ対応

```css
@media (max-width: 768px) {
  .progress-grid {
    grid-template-columns: 1fr; /* 1列に */
  }
}
```

## 🤔 よくある質問

### Q1: 左のグローバルナビゲーションはどうする？
**A**: 対象外です。右側のメインコンテンツのみ実装してください。

### Q2: カードコンポーネントも実装する？
**A**: 不要です。ProgressLesson、ArticleCard、FavoriteArticleCardは既に提供されているので、importして使ってください。

### Q3: タブの切り替えはどう実装する？
**A**: React のuseStateでアクティブなタブを管理し、条件付きレンダリングで表示内容を切り替えます。サンプルコード（MyPage.tsx）を参照してください。

### Q4: レスポンシブ対応は必須？
**A**: はい。デスクトップでは2列、スマホでは1列になるように実装してください。
