## 構成（2ブロック）

### 1) レッスンブロック

- **役割**: レッスン詳細ページへのリンク
- **要素**:
  - レッスンアイコン
  - レッスンタイトル
  - 進捗率バー
  - 進捗率（%）
- **状態**:
  - 通常
  - hover / active（Web想定なら）
  - loading（必要なら）

### 2) 次の記事ブロック

- **役割**: 完了済み記事の「次に読むべき記事」へのリンク
- **要素**:
  - 固定ラベル: `次へ👉️`
  - 次の記事タイトル
- **状態**:
  - 次の記事が存在する
  - 次の記事が存在しない（最終記事完了時など）
