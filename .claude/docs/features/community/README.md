# コミュニティ機能レビュー

**作成日**: 2025-02-20
**ブランチ**: feature/community-pages

---

## 概要

このドキュメントは、最近実装したコミュニティ機能（質問・フィードバック・お役立ち）のレビュー結果と改善タスクを管理します。

---

## 対象機能

| 機能 | 実装場所 | ステータス |
|------|----------|------------|
| 質問 | `src/pages/questions/` | レビュー待ち |
| フィードバック | `src/pages/feedbacks/` | レビュー待ち |
| お役立ち（ナレッジ） | `src/pages/knowledge/` | レビュー待ち |

---

## ドキュメント構成

```
.claude/docs/features/community/
├── README.md              ← このファイル（概要）
├── REVIEW-TASKS.md        ← 改善タスク一覧
├── questions/             ← 質問機能のレビュー詳細
│   └── REVIEW.md
├── feedbacks/             ← フィードバック機能のレビュー詳細
│   └── REVIEW.md
└── knowledge/             ← お役立ち機能のレビュー詳細
    └── REVIEW.md
```

---

## レビュー観点

### 1. UI/UX
- デザインの一貫性
- レスポンシブ対応
- ユーザビリティ

### 2. 機能性
- 必要な機能が揃っているか
- エッジケースの処理
- エラーハンドリング

### 3. パフォーマンス
- 読み込み速度
- キャッシュ戦略
- 不要な再レンダリング

### 4. コード品質
- コードの可読性
- 重複コードの有無
- 型定義の適切さ

---

## 次のステップ

1. 各機能のレビューを実施 → 個別のREVIEW.mdに記録
2. 改善タスクをREVIEW-TASKS.mdに集約
3. 優先度を設定して実装開始
