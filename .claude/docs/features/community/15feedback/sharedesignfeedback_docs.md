**🚀 BONO：思考の循環システム構築（2nd Week 仕様・タスク）**

## 1. 施策の全体像：『思考のビフォーアフター』循環

ユーザーに「思考」を強制し、それをプロ（あなた）が「思想の証明」として称賛することで、ブランドポジションを確立する。

| **施策**                   | **紐づく目標**                  | **狙い**                                                       |
| -------------------------- | ------------------------------- | -------------------------------------------------------------- |
| **応募フォームの刷新**     | 数値トラッキング / AI時代の実践 | 「思考停止」を排除し、プレイヤー転換率（目標30%）を計測する。  |
| **マニフェストブログ公開** | ブランド思想の定義              | AI時代のデザイン＝「思考」であることを市場に宣言する。         |
| **解説付き引用RT**         | Playful / 継続率改善            | ユーザーの成長を「旅」として演出し、最高級の成功体験を与える。 |
| **思想共鳴型の編集者募集** | クリエイティブ水準の維持        | あなたの思想を動画で「増幅」できるパートナーを獲得する。       |

---

## 2. 実装仕様

### ① 15分フィードバック応募フォーム

ユーザーが「便利なAI」から「自分の意思」へ目を向けるための検問。

### ② 解説付き引用ポストの「型」

ユーザーのnote/ツイートを、ブランドの「思想の証明」に昇華させる儀式。

- **構成:** `[ユーザーのAfter画像]` + `[プロの視点解説]`
- **解説内容:** 「この修正のここが、プロの現場で求められる『眼』です。AI時代にはこの判断力こそが価値になります」という、思想（思考の重要性）の裏付け。
