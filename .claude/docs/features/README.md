# 機能別ドキュメント

ユーザー向け機能の仕様書をまとめるフォルダ。

## フォルダ構造

```
features/
├── README.md          ← このファイル
├── progress/          ← 進捗管理機能（調査完了）
├── favorites/         ← お気に入り機能（実装済み）
│   └── README.md
├── history/           ← 閲覧履歴機能（実装済み）
│   ├── README.md
│   └── specification.md
└── email/             ← メール機能（実装準備中）
    ├── README.md
    └── specification.md
```

## 各機能のドキュメント構成（推奨）

```
<機能名>/
├── README.md          ← 概要・ステータス・次のアクション
├── specification.md   ← 機能仕様（ユーザーストーリー、要件）
├── database.md        ← DBスキーマ設計
└── ui-design.md       ← UI設計・画面遷移
```

## ステータス一覧

| 機能 | ステータス | 備考 |
|------|-----------|------|
| 進捗管理 | 🔍 調査完了 | RLS無効など重大問題あり |
| お気に入り | ✅ 実装済み | - |
| 閲覧履歴 | ✅ 実装済み | 最大50件保存、20件表示 |
| メール | 🔧 実装準備中 | Resend採用、Phase1から開始 |
