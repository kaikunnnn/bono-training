```typescript
interface Stage {
  number: number; // 1, 2, 3, 4
  title: string;
  subtitle?: string;
  duration: string; // "3日", "1週間", "2週間", "1ヶ月"
  goals: string[];
  contents: Content[];
  challenge?: Challenge;
}
```

```typescript
interface Content {
```

```typescript
interface Challenge {
```

## 7. 実装時の考慮点

### 7.1 レスポンシブ対応

- モバイルセーフエリア: `calc(env(safe-area-inset-bottom))`
- ハンバーガーメニュー対応
- カード幅の調整

### 7.2 アクセシビリティ

- 適切な見出し階層（h1 > h2 > h3）
- ボタンのaria-label
- 色コントラスト確保

### 7.3 パフォーマンス

- 画像遅延読み込み
- コンテンツの段階的表示
