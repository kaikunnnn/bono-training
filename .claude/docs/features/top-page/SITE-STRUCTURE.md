## 5. キーメッセージまとめ

### ミッション
```
すべての人に創造性の夜明けを。
```

### 価値提案
```
転職実績のある実践的なコースカリキュラムを提供する
デザインでより良い社会を実現したい人のための学習サービス
```

### 差別化ポイント
1. **60+社の転職実績** - 実績で証明
2. **プロの運営** - 10年経験のプロダクトデザイナー
3. **コミュニティ** - 1人じゃない、仲間と学べる
4. **無制限の質問対応** - 挫折しない
5. **動画+実践** - 見るだけじゃない、実際にやる

### ターゲットユーザー
1. 未経験からUI/UXデザイナーを目指す人
2. Web・グラフィックデザイナーからの転職希望者
3. ジュニアデザイナーのスキルアップ志向者
