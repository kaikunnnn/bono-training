## 1. Stripe.com レイアウト分析

### 1.1 ヘッダー構成

```
┌─────────────────────────────────────────────────────────────────────┐
│ [Logo]  サービス▼ ソリューション▼ 開発者▼ リソース▼    料金 サインイン [今すぐ始める] [営業に問い合わせ] │
└─────────────────────────────────────────────────────────────────────┘
```

- **左側**: ロゴ + メガメニュー（サービス、ソリューション、開発者、リソース）
- **右側**: 料金リンク + サインイン + 2つのCTAボタン
- **スタイル**: 白背景、固定ヘッダー、ホバーでドロップダウン

### 1.2 ヒーローセクション

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│              事業成長を支える金融インフラ                              │
│                                                                     │
│     何百万もの企業がStripeを利用して～説明文～                          │
│                                                                     │
│            [始める]  [Google で登録]                                 │
│                                                                     │
│  ──────────────────────────────────────────────────                 │
│  [企業ロゴ] [企業ロゴ] [企業ロゴ] [企業ロゴ] [企業ロゴ]                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

- **構成**: 大きなタイトル + サブテキスト + 2つのCTA + 企業ロゴカルーセル
- **背景**: 波状のグラデーションイラスト
- **特徴**: シンプル、余白多め、視線誘導が明確

### 1.3 セクション順序

| # | セクション名 | 内容 |
|---|------------|------|
| 1 | ヒーロー | タイトル + CTA + 企業ロゴ |
| 2 | モジュールグリッド | Bento Grid でサービス紹介（オンライン決済、サブスク等） |
| 3 | イベント告知 | Stripe Sessions など |
| 4 | 統計セクション | 通貨数、決済額、稼働率、サブスク数 |
| 5 | 企業規模別ソリューション | 大企業、スタートアップ、プラットフォーム向け |
| 6 | 開発者セクション | API、ドキュメント |
| 7 | ニュース/イベント | カルーセル形式 |
| 8 | CTA | 最終訴求 |
| 9 | フッター | リンク集 |

### 1.4 Bento Grid レイアウト（重要）

```
┌─────────────────────┬─────────────┬─────────────┐
│                     │             │             │
│   大きいカード       │  中カード    │  中カード    │
│   (2x2)             │  (1x1)      │  (1x1)      │
│                     │             │             │
├─────────────────────┼─────────────┴─────────────┤
│                     │                           │
│   中カード (1x2)     │    横長カード (2x1)        │
│                     │                           │
└─────────────────────┴───────────────────────────┘
```

- **特徴**: 不均等なグリッド、視覚的な興味を生む
- **カード内容**: アイコン + タイトル + 簡潔な説明 + リンク

### 1.5 統計セクション

```
┌─────────────────────────────────────────────────────────────────────┐
│   135+        1兆ドル+      99.999%      10億+                      │
│   通貨        年間決済額    稼働率        サブスク                    │
└─────────────────────────────────────────────────────────────────────┘
```

- **特徴**: 大きな数字、シンプルなラベル、信頼性訴求

## 2. bo-no.design コンテンツ分析

### 2.1 サービス概要

- **サービス名**: BONO
- **ミッション**: 「すべての人に創造性の夜明けを」
- **価値提案**: プロから学べる実践的UI/UXデザイン学習

### 2.2 コース/コンテンツ一覧

| カテゴリ | コンテンツ名 |
|---------|------------|
| ロードマップ | UI/UXデザイナー転職ロードマップ |
| ロードマップ | 1年目活躍ロードマップ |
| スキルアップ | UIビジュアル基礎 |
| スキルアップ | 情報設計基礎 |
| スキルアップ | UIデザイン入門 |
| レッスン | UI Typography（36本動画+4お題） |
| レッスン | 3構造 |
| レッスン | センス習得 |
| レッスン | ビジュアル制作 |
| レッスン | 情報設計 |
| レッスン | UXデザイン基礎 |
| キャリア | ポートフォリオ作成法 |
| キャリア | 面接対策 |
| キャリア | 企業選択ガイド |

### 2.3 ターゲットユーザー

1. 未経験からUI/UXデザイナーを目指す人
2. Web・グラフィックデザイナーからの転職希望者
3. ジュニアデザイナーのスキルアップ志向者

### 2.4 料金プラン

| プラン | 3ヶ月継続 | 月額 |
|--------|---------|------|
| スタンダード | ¥5,800/月 | ¥6,800/月 |
| フィードバック | ¥13,800/月 | ¥15,800/月 |

### 2.5 強み・差別化

1. **10年経歴プロが運営・開発** - 実務経験に基づくカリキュラム
2. **広告費ゼロ** - 質と価格を両立
3. **転職実績60社以上** - 成果の証明
4. **コミュニティサポート** - 仲間と学べる
5. **無制限の質問・相談** - 挫折しない

### 2.6 実績数値

- 転職実績: 60社以上
- コンテンツ: 36本以上の動画
- 運営者経歴: 10年
