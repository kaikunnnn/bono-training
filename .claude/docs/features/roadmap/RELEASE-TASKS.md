### グラデーションの色・定義

#### プリセット名（統一済み）

| プリセット名    | ロードマップ       | 色                          |
| --------------- | ------------------ | --------------------------- |
| `career-change` | UIUXデザイナー転職 | #412731 → #382D28 → #22202B |

→ #482B4B(100%) → #2A2C42(73%) → #141520(0%) と変化する内容に変更。100%が下の方でそこから上にグラデのイメージです。

| `ui-beginner` | UIデザイン入門 | #BB5F70 → #494717 + overlay 38% |
#494717-100% -> BB5F70(0%)に変化。もし%が逆なら変更して100%が下の方でそこから上にグラデのイメージです。

| `ui-visual` | UIビジュアル入門 | #304750 → #5D5B65 + overlay 20% |
→OK

| `info-arch` | 情報設計基礎 | #8D7746 → #214234 + overlay 30% |
→向きが逆だと思います。 #214234 → #8D7746

| `ux-design` | UXデザイン基礎 | #2F3F6D → #764749 → #E27979 → #F1BAC1 + overlay 40% |
→これも向きが逆かも。

#### 定義場所

- `src/types/sanity-roadmap.ts` → `GRADIENT_PRESETS`（Hero用）
- `src/components/roadmap/RoadmapCardV2.tsx` → `GRADIENTS`（Card用）
