## 面白くなる視点（interestingPerspectives）

**セクション説明**: ようこそデザインの世界へ。「自分でつくる」と、世界が違って見えるよ

- **トレースで「観察力」が育つ**: 普段使っているアプリのUIに「なぜこうなっているのか」を考える視点を獲得できます

- **デザインツールを扱える**: ビジュアル制作ツールが扱えるようになり、できることの幅が広がります
- **3週間で「作れる自分」になる**: 短期間で成果が出るから、モチベーションを保ちながら続けられます
