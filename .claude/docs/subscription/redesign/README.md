## 🚀 作業開始時の手順

1. **MASTER-PLAN.mdを開く**
   - 現在のフェーズと次のタスクを確認

2. **該当フェーズのドキュメントを開く**
   - 詳細な手順とチェックリストを確認

3. **作業実施**
   - ログを記録しながら進める

4. **進捗を更新**
   - MASTER-PLAN.mdの進捗状況を更新

---

## 📝 問題発見時の対応

1. **即座にMASTER-PLAN.mdに記録**
   - 問題リストセクションに追加

2. **優先度を判定**
   - 🔴 Critical / 🟡 Medium / 🟢 Low

3. **次のフェーズで対応**
   - Phase 2で優先順位順に修正
