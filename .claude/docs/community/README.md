# コミュニティ機能プロジェクト

**プロジェクト名**: BONOやっていることをみんなが参照できるようにしよう
**開始日**: 2026-02-18
**ステータス**: 🚀 開始

---

## 概要

BONOコミュニティで行われている活動（質問、フィードバック、ナレッジ）を
サイト上で参照・検索できるようにするプロジェクト。

---

## ターゲット

| ユーザー | 目的 |
|---------|------|
| 既存ユーザー | コミュニティで行われていることを知覚・検索したい |
| 検討ユーザー | BONOでやっていることを事前に確認したい |
| 運営（自分） | コミュニティ活動をプロセス的に外部公開したい |

---

## Pain（課題）

### 既存ユーザー
- すべての情報がSlackに残っており、後から見返せない
- 検索ができない
- 質問のハードルが高く感じてしまう
- どういうものを質問していいかわからない

### フィードバックユーザー
- 他の人のフィードバック内容を見れない
- 自分で確認項目を思考して学習できない
- 成長スピードが上がりづらい
