# BONO サービス概要

**最終更新**: 2026-04-21
**目的**: BONOがどういうサービスかを体系的に整理したドキュメント

---

## 1. サービス概要

| 項目 | 内容 |
|------|------|
| **サービス名** | BONO |
| **種別** | オンラインUI/UXデザイン学習プラットフォーム（SaaS） |
| **ミッション** | すべての人に創造性の夜明けを |
| **タグライン** | はじめよう キモチがうごく ものづくり |
| **URL** | bo-no.design（Webflow） + training サブドメイン（React/Vercel） |

BONOは、UI/UXデザインを体系的に学べるオンライン学習サービス。未経験からUI/UXデザイナーへのキャリアチェンジを中心に、実践的なスキル習得とプロからのフィードバックを組み合わせた学習体験を提供する。

---

## 2. ターゲットユーザー

### 主要ターゲット

| ペルソナ | 特徴 | ニーズ |
|---------|------|--------|
| **キャリアチェンジャー** | 他職種からUI/UXデザイナーへ転職希望 | 体系的な学習パス、転職実績の証明 |
| **ジュニアデザイナー** | 実務1-2年の若手デザイナー | スキルの体系的な底上げ、プロの指導 |
| **独学者** | デザインスクールに通えない/通わない人 | 手頃な価格、柔軟なペース |

### 市場

- 日本語圏のデザイン教育市場
- 高額なデザインスクール（10万円以上/月）の代替として位置づけ

---

## 3. コンテンツ構造

BONOのコンテンツは以下の5層構造で整理されている。

```
ロードマップ (学習の全体地図)
  └── ステップ (学習段階)
        └── レッスン (テーマ別コース)
              └── クエスト (課題単位)
                    └── アーティクル (個別の教材：動画/記事)
```

### 3.1 ロードマップ

ゴール別の学習パス。現在5つのロードマップが存在する。

| ロードマップ | 対象 | 期間目安 |
|-------------|------|---------|
| **UIUXデザイナー転職** | キャリアチェンジャー | 6ヶ月 |
| **UIデザイン入門** | Figma初心者 | 1ヶ月 |
| **UIビジュアル入門** | UI表現の基礎を学びたい人 | 1-2ヶ月 |
| **情報設計基礎** | 設計力を鍛えたい人 | 2ヶ月 |
| **UXデザイン基礎** | 課題解決型デザインを学びたい人 | 2ヶ月 |

### 3.2 レッスン

テーマ別の学習コース。カテゴリは3つ。

- **情報設計** — 使いやすいUIをつくるための設計
- **UI** — ビジュアルデザイン、タイポグラフィ等
- **UX** — ユーザーリサーチ、課題定義等

### 3.3 トレーニング

レッスンとは別の、より実践的なハンズオン形式のプログラム。

| タイプ | 内容 |
|--------|------|
| **Challenge** | デザイン課題に取り組む |
| **Skill** | 特定スキルの実践練習 |
| **Portfolio** | ポートフォリオ制作 |

### 3.4 ガイド

体系的な解説記事。4つのカテゴリ。

- **Career** — キャリアに関するアドバイス
- **Learning** — 学び方のガイド
- **Industry** — 業界知識
- **Tools** — ツール・技術

### 3.5 その他のコンテンツ

| コンテンツ | 説明 |
|-----------|------|
| **ブログ** | サービス運営からの発信 |
| **イベント** | ワークショップ・交流会 |
| **ナレッジ** | メンバーによる知見共有 |

## 4. 主要機能

### 4.1 学習系

| 機能 | ステータス | 説明 |
|------|-----------|------|
| レッスン閲覧 | ✅ 稼働中 | 動画+記事形式の教材閲覧 |
| ロードマップ | ✅ 稼働中 | ゴール別の学習パス表示 |
| トレーニング | ✅ 稼働中 | 実践的なハンズオン課題 |
| 進捗管理 | 🔍 調査完了 | レッスン/記事の完了追跡（RLS未設定の問題あり） |
| お気に入り | ✅ 実装済み | 記事のブックマーク保存 |
| 閲覧履歴 | ✅ 実装済み | 最大50件保存、20件表示 |
| 検索 | ✅ 稼働中 | レッスン・ガイド・記事の横断検索 |
| AI検索 | ✅ 稼働中 | デザインに関するAIチャット |

### 4.2 コミュニティ系

| 機能 | 説明 |
|------|------|
| Q&A（質問） | デザインに関する質問を投稿・回答 |
| フィードバック | プロによるデザインレビュー（15分セッション） |
| ナレッジベース | メンバーの知見共有 |
| ユーザーアウトプット | メンバーの制作物を共有・展示 |

### 4.3 ユーザー管理

| 機能 | 説明 |
|------|------|
| アカウント認証 | メール/パスワードによるサインアップ・ログイン |
| マイページ | 学習進捗・ブックマーク・履歴の統合ダッシュボード |
| サブスクリプション | Stripe連携のプラン管理 |
| オンボーディング | サブスク完了後の初回ガイド（アルファ版） |

## 5. 料金プラン

| プラン | 月額（3ヶ月） | 月額（1ヶ月） | 含まれるもの |
|--------|-------------|-------------|-------------|
| **Standard** | ¥5,800/月 | ¥6,800/月 | 全学習コンテンツ + トレーニング |
| **Feedback** | ¥13,800/月 | ¥15,800/月 | 全コンテンツ + プロのフィードバック |

### アクセス制御

- **無料**: 一部のレッスン・記事のみ閲覧可能
- **Standard以上**: 全学習コンテンツ、トレーニング、コミュニティ機能
- **Feedback**: Standard の全機能 + プロによるデザインレビュー

### 決済基盤

- **Stripe** — サブスクリプション管理・決済処理
- **Stripe Customer Portal** — 解約・プラン変更はStripe経由

## 6. 実績・差別化

| 項目 | 内容 |
|------|------|
| **転職実績** | 60人以上のキャリアチェンジ成功 |
| **運営経験** | 10年以上のプロフェッショナル経験 |
| **広告費** | ゼロ（オーガニック成長） |
| **価格** | 大手スクールの1/10以下 |

### 3つの価値の柱

1. **体系的学習** — ロードマップ → レッスン → クエストの段階的カリキュラム
2. **実践トレーニング** — 実際のプロジェクトシナリオに基づくハンズオン課題
3. **プロのフィードバック** — 現役デザイナーによる1対1のレビュー

- **bo-no.design** (Webflow) — LP・マーケティングサイト・ブランドページ

```
/ ......................... トップ（未ログイン→/top、ログイン済→/mypage）

/roadmaps/:slug ........... ロードマップ詳細
```
