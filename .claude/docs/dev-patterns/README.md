開発用UIパターンのドキュメント

- **[PATTERN-E-STYLE.md](./PATTERN-E-STYLE.md)** - パターンEのデザインシステム（カラー、フォント、コンポーネント）

## ページ構成

| ページ | ファイル | ステータス |
|--------|----------|------------|
| トップページ | [TOP-PAGE.md](./TOP-PAGE.md) | 実装済（PatternF） |
| ガイドページ | [GUIDE-PAGE.md](./GUIDE-PAGE.md) | 構成待ち |
| ロードマップ | ROADMAP.md | 未作成 |
| レッスン詳細 | LESSON-DETAIL.md | 未作成 |
| フィードバック詳細 | FEEDBACK-DETAIL.md | 未作成 |

## 使い方

1. `PATTERN-E-STYLE.md` のスタイルをベースに各ページを実装
2. 各ページの構成は個別のMDファイルに記載
3. 実装時は `/dev/xxx` のURLでプレビュー可能
