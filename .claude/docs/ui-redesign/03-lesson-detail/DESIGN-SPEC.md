## 現状

### ファイル構成
```
src/pages/LessonDetail.tsx    # メインページ

使用コンポーネント:
├── Layout                    # グローバルレイアウト
├── LessonHero                # ヒーローセクション（アイコン、タイトル、説明）
│   └── InfoBlock             # カテゴリ・プラン表示
├── LessonTabs                # タブ切り替え（コンテンツ/概要）
├── QuestList                 # クエスト一覧
│   └── QuestCard             # クエストカード
│       └── ContentItem       # 記事アイテム
└── OverviewTab               # 概要タブ
```

### 現在の構造
- **LessonHero**: グラデーション背景 + アイコン + タイトル + 説明 + スタートボタン
- **LessonTabs**:
  - コンテンツタブ: Quest一覧 → 各Questの記事リスト
  - 概要タブ: 目的と概要テキスト

### ルート
- `/lessons/:slug`
