## 現状

### ファイル構成
```
src/pages/ArticleDetail.tsx   # メインページ

使用コンポーネント:
├── ArticleSideNav            # 左サイドナビ（PC: 320px固定）
│   ├── LogoBlock             # ロゴ
│   ├── BackNavigation        # 戻るリンク
│   ├── LessonSection         # レッスン進捗表示
│   ├── QuestBlock            # クエスト一覧
│   └── ContentItem           # 記事リスト
├── MobileMenuButton          # モバイルメニューボタン
├── MobileSideNav             # モバイル用サイドナビ
├── HeadingSection            # タイトル、完了/お気に入りボタン
├── VideoSection              # 動画/サムネイル
├── TodoSection               # 学習目標
├── RichTextSection           # 記事本文
├── ContentNavigation         # 前後ナビ
├── QuestCompletionModal      # クエスト完了モーダル
└── CelebrationModal          # レッスン完了モーダル
```

### 現在の構造
- **2カラムレイアウト**:
  - 左: ArticleSideNav（320px, sticky）
  - 右: メインコンテンツ（最大720px）
- **ビデオセクション**: 1680px以上で最大1320px
- **モバイル**: ハンバーガーメニューでSideNav開閉

### ルート
- `/articles/:slug`

### 特徴
- 完了チェック、ブックマーク、シェア機能
- クエスト/レッスン完了時のモーダル表示
- 前後記事へのナビゲーション
