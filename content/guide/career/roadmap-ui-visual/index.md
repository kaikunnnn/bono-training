# UIデザインの見た目の基本スキルを身につけるために必要なこと

「なんとなく良いデザイン」から「なぜ良いか説明できるデザイン」へ。このガイドでは、UIの見た目を作るための**基本スキル**を解説します。

## なぜ「見た目の基本」が重要なのか？

UIデザイナーとして働くとき、必ず問われるのが**「なぜこのデザインにしたのか？」**という質問です。

- なぜこのサイズにしたのか？
- なぜこの色を選んだのか？
- なぜこの配置にしたのか？

これに答えられないと、デザインの修正依頼に振り回されます。逆に、**根拠を持ってデザインできる**と、説得力のある提案ができるようになります。

## UIビジュアルの6つの基本要素

### 1. サイズと余白

UIデザインで最も重要な要素の一つです。

**身につけるべきこと**：
- 8の倍数ルール（4px, 8px, 16px, 24px...）
- 余白の使い分け（タイト/ルーズ）
- 視覚的な階層の作り方

### 2. タイポグラフィ

文字の扱い方がUIの印象を大きく左右します。

**身につけるべきこと**：
- フォントサイズの決め方
- 行間・字間の調整
- 見出しと本文のコントラスト

### 3. 色とカラーシステム

色の使い方には明確なルールがあります。

**身につけるべきこと**：
- プライマリ/セカンダリカラーの設定
- 背景色とテキスト色のコントラスト
- 状態を表す色（成功/エラー/警告）

### 4. コンポーネント設計

再利用可能なUI部品を作る力です。

**身につけるべきこと**：
- ボタンのバリエーション
- フォーム要素の設計
- カードとリストの構造

### 5. レイアウト

要素の配置の仕方です。

**身につけるべきこと**：
- グリッドシステム
- 画面構成のパターン
- レスポンシブ対応の考え方

### 6. アイコンとイメージ

視覚的な要素の扱い方です。

**身につけるべきこと**：
- アイコンの選び方とサイズ
- 画像の配置とアスペクト比
- イラストやグラフィックの使い方

## 「センス」は後からついてくる

よくある誤解があります。

> 「デザインはセンスだから、自分には向いてない」

これは間違いです。実際は：

1. **基本ルールを覚える**
2. **良いデザインを観察する**
3. **実際に作って試す**
4. **フィードバックを受けて改善する**

このサイクルを繰り返すことで、「センス」は養われます。

## ビジュアルスキルの次のステップ

見た目を作れるようになったら、次は**「使いやすさ」の設計**です。

- 情報の整理と優先順位
- 画面遷移とナビゲーション
- ユーザーの行動を考えた設計

これが「情報設計」のスキルです。見た目と使いやすさ、両方を兼ね備えたUIデザイナーを目指しましょう。
