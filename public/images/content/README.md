```

├── career/     # キャリアセクション用画像
├── ux/         # UXセクション用画像
└── ui/         # UIセクション用画像
```

## 画像仕様

- **フォーマット**: JPG, PNG, WebP推奨
- **アスペクト比**:
  - ガイドカード（大）: 8:5 (横長)
  - 読みものカード: 3:2 (横長)
- **ファイルサイズ**: 200KB以下推奨（Web最適化）
- **色空間**: sRGB

## 画像が未配置の場合

画像ファイルが配置されていない場合、自動的にフォールバック絵文字が表示されます。

- career-guide.jpg → 📚
- interviews.jpg → 👩‍💻
- feedback.jpg → 💬
- outputs.jpg → 📝
