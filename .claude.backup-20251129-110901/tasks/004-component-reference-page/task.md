# タスク: コンポーネント・カラーリファレンスページ

## タスク情報

- **タスクID**: 004
- **タスク名**: コンポーネント・カラーリファレンスページ
- **作成日**: 2025-10-31
- **ステータス**: 進行中
- **優先度**: 中

## 概要

開発環境でのみアクセス可能な、サイト全体のデザインシステムとコンポーネントライブラリを一覧できるリファレンスページを構築する。

## 要求事項

ユーザーからの要求:
> サイトで定義されてるコンポーネントと色定義をページで確認できるページを作って欲しいです。dev状態だけで確認できるといいな

## 目標

1. すべてのコンポーネント(shadcn/ui + カスタム)を視覚的に確認できる
2. カラーパレット、タイポグラフィ、アニメーションを一覧表示
3. 開発環境でのみアクセス可能
4. 各コンポーネントの使用例を表示
5. デザイントークンの一元管理

## 成果物

- `/dev/components` ページ
- カラーパレット表示セクション
- タイポグラフィサンプル表示
- コンポーネントライブラリ表示
- アニメーションサンプル表示

## 制約事項

- 開発モード (`import.meta.env.DEV`) でのみアクセス可能
- 本番ビルドには含めない、または404にリダイレクト
