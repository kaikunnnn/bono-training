# タスク: ガイドページの実装

## タスク概要

学習ガイド記事を管理・表示するための `/guide` ページを実装する。
転職ロードマップ、AIとデザイナーの未来、学習方法など、記事コンテンツベースの学習ガイドを提供する。

## 目的

- ユーザーに体系的な学習ガイドを提供
- カテゴリ別に整理された記事一覧を表示
- WikiやKnowledge Baseのように情報を集約

## 対象ページ

- `/guide` - ガイド一覧ページ（カテゴリ別表示）
- `/guide/:slug` - 個別ガイド記事詳細ページ
